# Teslo API

1. Clonar proyecto

2. Instalar las dependencias
```
npm install
```

3. Renombrar el archivo __.env.template__ a __.env__

4. Cambiar las variables en __.env__

5. Levantar la base de datos

```
docker-compose up -d
```

6. Correr la app en modo desarrollo
```
npm run start:dev
```
